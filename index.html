    Welcome to mosesDB!

    The simple, small-scale database for non-critical data

    To get started:

    1. Download the repository.
    2. Unpack the repository.
    3. Run ./compile
    4. Create a schema(template) like the one provided
    5. Enjoy!

    mosesDB has NO DEPENDENCIES and very little reliance on the C standard library!

    Feel free to contribute, there is still much that needs to be done!
